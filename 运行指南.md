# cn2an 项目运行指南

## 一、环境准备

### 1. 检查 Python 版本
```bash
python --version
# 需要 Python 3.6 或更高版本
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

主要依赖：
- `proces>=0.1.7` - 数据预处理管道

## 二、安装项目

### 方式一：开发模式安装（推荐）
```bash
python setup.py develop
```
或
```bash
pip install -e .
```

### 方式二：直接安装
```bash
python setup.py install
```

## 三、运行测试

### 1. 运行所有测试
```bash
python -m unittest discover -s cn2an -p "*_test.py"
```

### 2. 运行特定测试文件
```bash
# 测试 transform 功能
python -m unittest cn2an.transform_test

# 测试 cn2an 功能
python -m unittest cn2an.cn2an_test

# 测试 an2cn 功能
python -m unittest cn2an.an2cn_test
```

## 四、使用示例

### 1. 基本使用（Python 交互式）

```python
# 启动 Python
python

# 导入模块
import cn2an

# 查看版本
print(cn2an.__version__)

# 中文数字转阿拉伯数字
result = cn2an.cn2an("一百二十三")
print(result)  # 输出: 123

# 阿拉伯数字转中文数字
result = cn2an.an2cn("123")
print(result)  # 输出: 一百二十三

# 句子转换（测试修复后的功能）
result = cn2an.transform("1-2个月", "an2cn")
print(result)  # 输出: 一到二个月
```

### 2. 创建测试脚本

创建文件 `test_demo.py`：

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
import cn2an

# 测试中文数字转阿拉伯数字
print("=== 中文数字转阿拉伯数字 ===")

print(cn2an.an2cn("123", "up"))  # 壹佰贰拾叁

# 测试句子转换
print("\n=== 句子转换 ===")
print(cn2an.transform("小王捡了一百块钱", "cn2an"))  # 小王捡了100块钱
print(cn2an.transform("小王捡了100块钱", "an2cn"))  # 小王捡了一百块钱

print(cn2an.transform("3-5年", "an2cn"))  # 三到五年
print(cn2an.transform("10-20个", "an2cn"))  # 十到二十个
```

## 五、常用功能示例

### 1. 中文数字转阿拉伯数字

```python
import cn2an

# strict 模式（默认）
cn2an.cn2an("一百二十三")  # 123

# normal 模式
cn2an.cn2an("一二三", "normal")  # 123

# smart 模式（支持混合）
cn2an.cn2an("1百23", "smart")  # 123

# 支持负数
cn2an.cn2an("负一百二十三")  # -123

# 支持小数
cn2an.cn2an("一点二三")  # 1.23
```

### 2. 阿拉伯数字转中文数字

```python
import cn2an

# low 模式（默认，小写）
cn2an.an2cn("123")  # 一百二十三

# up 模式（大写）
cn2an.an2cn("123", "up")  # 壹佰贰拾叁

# rmb 模式（人民币）
cn2an.an2cn("123", "rmb")  # 壹佰贰拾叁元整

# 支持负数
cn2an.an2cn("-123")  # 负一百二十三

# 支持小数
cn2an.an2cn("1.23")  # 一点二三
```

### 3. 句子转换

```python
import cn2an

# 中文数字转阿拉伯数字
cn2an.transform("小王捡了一百块钱", "cn2an")
# 输出: 小王捡了100块钱

cn2an.transform("今天股价上涨了8%", "an2cn")
# 输出: 今天股价上涨了百分之八

# 阿拉伯数字转中文数字
cn2an.transform("小王捡了100块钱", "an2cn")
# 输出: 小王捡了一百块钱

# 修复后的数字范围功能
cn2an.transform("1-2个月", "an2cn")
# 输出: 一到二个月（修复前是"一负二个月"）
```

## 六、快速测试修复功能

创建一个快速测试脚本 `quick_test.py`：

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
import cn2an

print("测试数字范围转换修复：")
print("-" * 40)

test_cases = [
    ("1-2个月", "一到二个月"),
    ("3-5年", "三到五年"),
    ("10-20个", "十到二十个"),
]

for input_str, expected in test_cases:
    result = cn2an.transform(input_str, "an2cn")
    status = "✓" if result == expected else "✗"
    print(f"{status} 输入: {input_str}")
    print(f"   期望: {expected}")
    print(f"   实际: {result}")
    print()
```

运行：
```bash
python quick_test.py
```
